# ZonaOscura

Prototipo responsive para reportar, visualizar y gestionar zonas con deficiente iluminación pública.

## Funcionalidades principales

- Registro y seguimiento de reportes ciudadanos.
- Selección de ubicación en mapa y evidencia fotográfica.
- Visualización de zonas oscuras geolocalizadas.
- Historial municipal de zonas atendidas.
- Priorización de casos críticos.
- Generación de informes operativos.

## Desarrollo

```bash
pnpm install
pnpm dev
```

## Validación

```bash
pnpm build
```
